# Example Block
This is a boilerplate/template block which is intended to be a guide to create further blocks. The Example block uses Wordpress native block options, Advanced Custom Fields Pro and a SCSS compiler. Follow instructions in /blocks/ readme file.

## Using the example block
1. Copy the example block folder and rename as your new block.
2. Edit file names and delete any files not required.
3. Delete any features you do not need.
