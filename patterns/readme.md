# Patterns

A pattern is a block or many blocks which are reusable throughout the website. All patterns used for any blocks will be in this folder and must end with '-pattern.php'. 

## Creating a pattern

1. Make a copy of the example-pattern.php file -> rename it appropriately eg. mypattern-pattern.php
2. Plugin file location /patterns/ & child theme location 'child-theme/braftonium/patterns' (create folders if they don't exist)
3. Edit pattern Title and Description
4. Create a block layout in Wordpress back end on a page
5. Copy all the blocks you want & paste them in place of the current demo 'content'
